Neuen Gerätetyp erstellen

Eine Gerätetyp definiert das Gerät mit allen dazugehörigen Feldgruppen. Der PC ist ein Gerätetyp mit mehreren Feldgruppen. Die einzelnen Feldgruppen können in dem Menüpunkt "Eigene Felder" genauer definiert werden. Nach der Fertigstellung eines Gerätetyps können nun unter "Geräte hinzufügen" neue Geräte dieses Typs hinzugefügt werden.
