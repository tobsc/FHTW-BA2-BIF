Neue Feldgruppe erstellen

Eine Feldgruppe definiert eine Komponente eines Geräts. Der Display eines PCs wäre die Feldgruppe Display. Abzählbar definiert, ob diese Komponente öfters als einmal am Gerät vorhanden sein kann.
Bsp: Ein Laptop hat genau einen Display aber vielleicht mehrere USB-Anschlüsse.
